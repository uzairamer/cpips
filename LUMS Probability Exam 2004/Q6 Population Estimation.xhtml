There are \(N\) deer in a forest. We capture $Y2 of them, mark them, and then release them again. Later we capture $Y3 and find that $Y4 of the $Y3 are marked. Estimate the number of deer in the forest assuming mixing, no change in the number between capture and recapture, size of original captured group large enough to reperesent population

What is the value of \(N\)

Explanation:

Mark and recapture is a method commonly used in ecology to estimate an animal population's size. A portion of the population is captured, marked, and released. Later, another portion is captured and the number of marked individuals within the sample is counted.

n = Number of animals marked on the first visit $Y2

K = Number of animals captured on the second visit $Y3

k = Number of recaptured animals that were marked $Y4

N = Number of animals in the population = K*n/k = $A_0
